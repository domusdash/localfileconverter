# Blueprint Converter - Local File & Image Converter

A premium, local-first file and image converter web application. Convert PDF, HEIC, WebP, PNG, JPG, and SVG files entirely client-side. Your files never leave your computer.

## Features

- **100% Client-Side Processing:** No server uploads. Absolute privacy and data security.
- **HEIC to JPG/PNG/WebP:** Convert iOS HEIC photos in bulk.
- **PDF to JPG/PNG/WebP:** Extract specific pages or entire documents as image files.
- **Cross-Image Converter:** Convert WebP, PNG, JPG, BMP, TIFF, SVG, and more.
- **Batch ZIP Downloads:** Convert multiple files simultaneously and download them in a single ZIP.
- **Image Settings:** Adjustable quality/compression slider and dimensions scaling.
- **PDF Rendering Quality:** Select resolution options (72 DPI, 150 DPI, 300 DPI).

## Local Development

Simply open `index.html` in any web browser, or run a local development server:

```bash
npx serve .
```
